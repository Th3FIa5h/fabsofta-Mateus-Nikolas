### Revisão de Java

 - JRE - Java Runtime Environment (java.exe)
    - Máquina Virtual do Java - Interpreta o Bytecode (.class) do Java para linguagem de máquina
 - JDK - Java Development Kit (javac.exe)
  - Compilar o código Java em bytecode (.class)
